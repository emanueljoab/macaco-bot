1. Introduction 1. Introdução

This Privacy Policy describes how Gerador de Macaco Aleatório collects, uses, and stores data from Discord users and servers. By adding or using the bot, you agree to the practices described in this document.

Esta Política de Privacidade descreve como o Gerador de Macaco Aleatório coleta, utiliza e armazena dados de usuários e servidores do Discord. Ao adicionar ou utilizar o bot, você concorda com as práticas descritas neste documento.

2. Data Collected 2. Dados coletados

The bot stores only the data strictly necessary for its operation:

O bot armazena apenas os dados estritamente necessários para seu funcionamento:

  • Discord user ID and username — for rankings and match history.
  • Server (guild) ID and name — to separate data between different servers.
  • Game statistics — wins/losses in rock-paper-scissors and personal records for entertainment commands (howgay, pp, simp, stank).
  • Server language preference — Portuguese or English.
  • ID e nome de usuário do Discord — para rankings e histórico de partidas.
  • ID e nome do servidor (guild) — para separar dados entre servidores distintos.
  • Estatísticas de jogo — vitórias/derrotas no jokenpo e recordes pessoais dos comandos de entretenimento (howgay, pp, simp, stank).
  • Preferência de idioma do servidor — português ou inglês.

3. Use of Data 3. Uso dos dados

The collected data is used solely for:

Os dados coletados são utilizados somente para:

  • Displaying rankings and match history within the server.
  • Persisting personal records between sessions.
  • Applying the language preference configured by the server.
  • Exibir rankings e histórico de partidas dentro do servidor.
  • Persistir recordes pessoais entre sessões.
  • Aplicar a preferência de idioma configurada pelo servidor.

Data is not sold, shared, or used for commercial or advertising purposes.

Os dados não são vendidos, compartilhados nem utilizados para fins comerciais ou publicitários.

4. Retention and Deletion 4. Retenção e exclusão

Data is kept while the bot is present in the server or while the user continues to interact with it. To request deletion of your data, contact us through the channels below.

Os dados são mantidos enquanto o bot estiver presente no servidor ou enquanto o usuário continuar interagindo com ele. Para solicitar a exclusão dos seus dados, entre em contato pelos canais abaixo.

5. Security 5. Segurança

Data is stored locally in an SQLite database on the bot host server. No sensitive information (tokens, passwords, emails) is collected.

Os dados são armazenados localmente em banco de dados SQLite no servidor de hospedagem do bot. Nenhuma informação sensível (tokens, senhas, e-mails) é coletada.

6. Changes to This Policy 6. Alterações nesta política

This policy may be updated periodically. The date of the last update will always be indicated at the top of the page.

Esta política pode ser atualizada periodicamente. A data de última atualização sempre será indicada no topo da página.
